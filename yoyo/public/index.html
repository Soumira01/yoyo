📖 Comment jouer ?

🎯 Objectif

Former un maximum de carrés parfaits en plaçant des points sur les intersections de la grille.

📌 Tour de jeu

  • Les joueurs jouent chacun à leur tour
  • Clique sur une intersection pour placer un point
  • Tu as 8 secondes par tour
  • Si tu ne joues pas → le tour passe à l'adversaire

⬜ Former un carré

  • 4 points de ta couleur forment un carré parfait aligné sur la grille
  • Tous les carrés ont la même taille (1 case)
  • Chaque carré vaut 1 point
  • Un même point peut contribuer à plusieurs carrés

🗺️ Navigation

  • 📱 Mobile : pinch pour zoomer, glisse pour déplacer
  • 💻 PC : molette pour zoomer, clic droit/milieu pour déplacer
  • Bouton 🏠 pour recentrer la vue

⏱️ Fin de partie

La partie dure 10 minutes. Le joueur avec le plus de carrés gagne !
